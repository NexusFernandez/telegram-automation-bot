# Telegram AI Business Bot

**Python Telegram bot automation toolkit for lead scraping, notifications, and workflow automation.**

A practical Telegram command bot for operators, freelancers, and founders who want a private control panel inside Telegram.

## Features

- /report — Generate daily/weekly business reports
- /schedule — Schedule reminders and tasks
- /alert — Set up automated alerts and notifications
- /memo — Quick note-taking with organization
- /ai — AI-powered responses via Ollama integration

## Why this bot?

- **Simple to understand** — Clear commands, no learning curve
- **Useful on day one** — Deploy and start using immediately
- **No SaaS dependency** — Runs on your own infrastructure
- **Easy to customize** — Clean Python code, well-documented
